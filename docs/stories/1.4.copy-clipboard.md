## Story

**As a** user,  
**I want** to copy the generated message with one click,  
**so that** I can paste it directly into my Git client.

## Acceptance Criteria

1. [x] Botão "Copiar" copia a mensagem completa (FR5).
2. [x] Feedback visual e `aria-live="polite"` anunciam sucesso ou falha.
3. [x] Botão desabilitado quando formulário inválido.
4. [x] Teste unitário ou integração do fluxo de cópia (mock de clipboard).
