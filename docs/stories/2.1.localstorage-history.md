## Story

**As a** user,  
**I want** my generated messages saved locally,  
**so that** I can refer back to recent commits without retyping.

## Acceptance Criteria

1. Cada cópia bem-sucedida salva entrada com id, timestamp, campos e mensagem (FR6).
2. Máximo 50 entradas; FIFO ao exceder (NFR6).
3. Histórico sobrevive refresh da página.
4. Tratamento gracioso se `localStorage` indisponível.
5. Testes unitários para append, limit e parse.
