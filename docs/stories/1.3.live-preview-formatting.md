## Story

**As a** user,  
**I want** to see the formatted commit message update as I type,  
**so that** I know exactly what will be copied.

## Acceptance Criteria

1. [x] Preview exibe `type(scope): description` ou `type: description` conforme FR4.
2. [x] Atualização em tempo real sem reload (FR10).
3. [x] Função pura `buildCommitMessage(type, scope, description)` coberta por testes unitários.
4. [x] Aviso não bloqueante se descrição começar com maiúscula ou terminar com ponto.
